## Project 11: Visual Inspection of Connecting Rods
*Project by Vincenzo Lombardi.*<br><br>
This folder contains the material related to the development (in its choices and final results) of *a software system aimed at visual inspection of motorcycle connecting-rods.* For a detailed description of it, please refer to the content of the [Jupyter Notebook][1] file **[project11VincenzoLombardi.ipynb][2]**.<br>
Note that the current folder also includes the following materials:
- `miscellaneous.py`: a Python file that contains auxiliary functions / data structures (used within the Jupyter Notebook file)
- `plotter.py`: a Python file dedicated to plotting/visualization (used within the Jupyter Notebook file)
- `parameters.py`: a Python that contains all the parameters the project is based onto (used within the Jupyter Notebook file)
- `data/images/FirstTask` folder: a folder that contains images used in the First Task of the Project (named with increasing natural numbers)
- `data/images/SecondTask` folder: a folder that contains images used in the Second Task of the Project (named with increasing natural numbers)

[1]: https://jupyter.org/
